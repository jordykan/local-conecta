# Instrucciones

Para generar los iconos PWA, puedes usar:

1. **pwa-asset-generator** (automático):
   ```bash
   npx pwa-asset-generator [tu-logo.png] public/icons --icon-only --background '#ea580c' --type png
   ```

2. **Manualmente**: Crea iconos PNG en estos tamaños:
   - 72x72
   - 96x96
   - 128x128
   - 144x144
   - 152x152
   - 192x192
   - 384x384
   - 512x512

3. **Screenshots**: Crea capturas de pantalla de la app:
   - screenshot-mobile.png (540x720)
   - screenshot-desktop.png (1280x720)

Los iconos deben estar en formato PNG con fondo transparente o el color de marca #ea580c.
